Application web de configuration de guitare personnalisée.

Fonctionnalités :

- Connexion utilisateur
- Création de configurations
- Modification et suppression des configurations
- Personnalisation des composants de la guitare
- Enregistrement des commandes en base de données

Technologies :

- C#
- ASP.NET Razor
- SQL Server
- API

Objectif : Développer une application web complète avec gestion utilisateur et personnalisation de données.
